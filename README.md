# ProductSearch

Product Search system using MVC. This system accesses the database and displays the searched product on the screen. Without logging in, the user can only search for products. After logging in, the user can update products, add new products and delete products. If the user is Admin, he/she can go to the dashboard page and see data such as who updated the product last time, number of products, etc. Admin can add new admin and add new users.
